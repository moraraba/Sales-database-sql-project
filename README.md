# Sales-database-sql-project
A relational database project built with SQL that includes schema creation, sample data, and analytical queries on sales transactions.

# Database Structure
The project creates the following tables:
- **Customers** – Stores customer details and membership status.
- **Products** – Contains product information such as price and category.
- **Sales** – Records transactions between customers and products.
- **Inventory** – Tracks the available stock for each product.
 
# Objectives
The purpose of this assignment is to practice:
 - Creating database schemas and tables
 - Inserting sample data into tables
 - Writing SQL queries to retrieve and analyze data
 - Using joins, aggregate functions, and filtering conditions
